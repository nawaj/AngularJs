This Directive is used to show the content when hover any parent <div> or any parent tag.
Just add this "showonhoverparent" in your content hover tag.

Example:

<div class="parenttag"><!--when you hover the mouse to this div below div content will get visible-->
<div class="hovertag" style="display:none" showonhoverparent>
<p>AwesomeM</p>
</div>
</div>
